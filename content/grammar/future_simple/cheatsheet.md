| Person | Conjugation | English |
| :--- | :--- | :--- |
| εγώ | **θα γράψω** | I will write |
| εσύ | **θα γράψεις** | You will write |
| αυτός/ή/ό | **θα γράψει** | He/She/It will write |
| εμείς | **θα γράψουμε** | We will write |
| εσείς | **θα γράψετε** | You (all) will write |
| αυτοί/ές/ά | **θα γράψουν** | They will write |

* **-π-, -β-, -φ-, -πτω** → change to **-ψ-** (e.g., *γράφω* → *θα γράψω*)
* **-κ-, -γ-, -χ-, -χνω** → change to **-ξ-** (e.g., *ανοίγω* → *θα ανοίξω*)
* **-τ-, -δ-, -θ-, -ζω** → change to **-σ-** (e.g., *διαβάζω* → *θα διαβάσω*)
