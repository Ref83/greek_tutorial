# Content vocabulary block page

Vocabulary block should consist of the list of items. 
For each Item when user click on it, it should expand the detailed information. 
Detailed information should consist of: 
    - `item` - the vocabulary item in Greek
    - `meaning` - meaning described in English
    - `examples` - the list of examples. Each example consist of `example` - example in Greek, and `translation` - translation in English
    - `colocations` - the list of colocation. Each colocation consist of `collocation` - collocation in Greek, and `translation` - translation in English

On the top of the page should be buttons: 
    - `Exercises` that lead to the page that described in `vocabulary_exercises.md`
    - `Read` that lead to the page that described in `vocabulary_read.md`
