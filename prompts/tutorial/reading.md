# Content grammar section page

The grammar page should show grammar topics. 
The list of grammar topics located in `content\grammar\topics.json` file.
Each grammar topic item should have an level mark from `content\grammar\topics.json`
On click of topic the user should be forwarded on topic page with content that described in `grammar_topic.md`.
The grammar page should have button "Exercises" on the top. When user this button the system should ask user to chose topics. And then run exercises from all selected topics like described in `grammar_topic.md` in random order.
