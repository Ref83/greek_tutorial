# Content vocabulary item page

The page should consist of data from item from `vocabulary.json` file
The header of the page should be from `phrase`
Then should be fields and sections `meaning`, `forms`, `usage_patterns`, `collocations`, `additional_examples`, and `notes`

At the top of the page should be the panel with the icon on exercises.
When user click on this button the page exercises that have `item_id` is equal to the current item should be opened.
The behavior of exercises page described in `vocabulary_exercises.md`
