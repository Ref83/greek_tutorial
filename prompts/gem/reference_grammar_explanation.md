# Grammar explanation template

## Overview
This reference explain the format of grammar explanation documents.

---

## Grammar explanation should include

- Grammar topic name: the name of grammar topic
- The Concept: A short summary of the grammar topic.
- Usage: The use cases when the grammar topic is used.
- The Mechanics: The deep explanation of the grammar topic. With a lot of examples for each part of topic. If it applicable the definition should contain the positive, negative and question form of grammar.
- Exceptions: The exceptions of the grammar topic with examples.

## Example

# Noun gender

## The Concept
In Greek, **every noun has a gender**, even objects and ideas.

There are **three genders**:

- **Masculine (ο)**
- **Feminine (η)**
- **Neuter (το)**

### Masculine Nouns
Usually end in: -ος, -ας, -ης
Definite Articles: ο (single), οι (plural)
Indefinite Article: ένας

**Creating plural**
To create plural masculine nouns change the end:
- -ος → -οι
- -ας, -ης → -ες

**Examples**
- **ο φίλος** – the friend
- **ο πατέρας** – the father
- **ένας μαθητής** – a student
- **ο άντρας** – the man
- **ο δάσκαλος** – the teacher
- **οι δρόμοι** – the roads
- **ο γιατρός** – the doctor
- **ο αδελφός** – the brother
- **ο χρόνος** – the time/year
- **ο κόσμος** – the world/people

### Feminine Nouns
Usually end in: -α, -η
Definite Articles: η (single), οι (plural)
Indefinite Article: μία / μια

**Creating plural**
To create plural feminine nouns change the end:
- -α, -η → -ες

**Examples**
- **μία φίλη** – a friend
- **η μητέρα** – the mother
- **η τάξη** – the class
- **οι γυναίκες** – the women
- **η πόλη** – the city
- **η θάλασσα** – the sea
- **η οικογένεια** – the family
- **η νύχτα** – the night
- **η ζωή** – the life
- **η εργασία** – the work/job

### Neuter Nouns
Usually end in: -ο, -ι, -μα
Definite Articles: το (single), τα (plural)
Indefinite Article: ένα

**Creating plural**
To create plural neuter nouns change the end:
- -ο → -α
- -ι → -ια
- -μα → -ματα

**Examples**
- **τα βιβλία** – the books
- **το παιδί** – the child
- **το μάθημα** – the lesson
- **το σπίτι** – the house
- **το σχολείο** – the school
- **ένα αυτοκίνητο** – a car
- **το όνομα** – the name
- **το φως** – the light
- **το γράμμα** – the letter
- **το τραπέζι** – the table

## Exceptions
Some nouns can have endings from one category but have different gender 

### Here the list of examples
- **η οδός** – the street (Feminine, but ends in -ος)
- **η διάμετρος** – the diameter (Feminine, but ends in -ος)
- **η νήσος** – the island (Feminine, but ends in -ος)
- **η ψήφος** – the vote (Feminine, but ends in -ος)
- **η άμπελος** – the vine (Feminine, but ends in -ος)
- **το τέλος** – the end (Neuter, but ends in -ος)
- **το μέρος** – the place/part (Neuter, but ends in -ος)
- **το έδαφος** – the ground (Neuter, but ends in -ος)
- **το κράτος** – the state (Neuter, but ends in -ος)
